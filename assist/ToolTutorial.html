Pandas 数据处理库

Pandas 是 Python 中用于数据处理和分析的强大库，提供了高效的 DataFrame 数据结构和丰富的数据操作功能。

核心功能

数据结构

提供 Series（一维）和 DataFrame（二维）数据结构，灵活存储和处理各类数据。

数据清洗

提供缺失值处理、重复值去除、数据转换等完整的数据清洗功能。

数据操作

支持数据筛选、排序、分组、合并等多种数据操作，满足复杂分析需求。

快速入门

基本操作示例

import pandas as pd

# 创建示例数据
data = {
'姓名': ['张三', '李四', '王五', '赵六'],
'年龄': [22, 25, 23, 24],
'专业': ['计算机', '数学', '统计学', '计算机']
}

# 创建DataFrame
df = pd.DataFrame(data)
print(df)

数据筛选操作：

# 筛选计算机专业的学生
cs_students = df[df['专业'] == '计算机']
print(cs_students)

# 筛选年龄大于23岁的学生
older_students = df[df['年龄'] > 23]
print(older_students)

NumPy 数值计算库

NumPy 是 Python 科学计算的基础库，提供了高性能的多维数组对象和数学函数库。

核心功能

多维数组

ndarray 数据结构，支持高效的数值运算和广播功能。

数学函数

提供大量数学运算函数，包括线性代数、傅里叶变换等。

随机数生成

强大的随机数生成工具，支持多种概率分布。

快速入门

基本操作示例

import numpy as np

# 创建数组
arr = np.array([1, 2, 3, 4, 5])
print(arr)

# 数组运算
print(arr + 10)
print(arr * 2)

《A Recursive Subdivision Technique for Sampling Multi-class Scatterplots》

基于定制二叉 kd 树的非均匀递归采样技术，可在多类散点图中忠实呈现相对数据和类别密度，同时保留主要异常值，经定量评估和案例研究验证其效果优于现有方法。

《Crowdsourcing Graphical Perception: Using Mechanical Turk to Assess Visualization Design》

在 Amazon Mechanical Turk 上复制过往图形感知实验并开展新实验，验证了众包用于图形感知研究的可行性，得出可视化设计新见解并给出众包研究设计建议。

核心文献

《Table-GPT: Table Fine-tuned GPT for Diverse Table Tasks》

提出 “表格微调” 范式，对 GPT-3.5 和 ChatGPT 等语言模型进行微调，使其在多种表格任务中表现优于原始模型，且具备良好的泛化能力。

核心文献

《A Relational Model of Data for Large Shared Data Banks》

提出基于 n 元关系、数据基关系范式和通用数据子语言的关系数据模型，解决了现有模型的数据依赖问题，为大型共享数据库提供了数据独立性和一致性保障的基础方法论。

专题文献

《A Survey on the Detection Algorithms for False Data Injection Attacks in Smart Grids》

系统梳理了智能电网中虚假数据注入攻击的检测算法，将其分为模型驱动和数据驱动两类，分析了各类算法的优缺点、适用场景及性能，指出了未来算法开发的核心准则。

从基础数学知识（标准差、协方差、特征向量等）入手，详细讲解了主成分分析（PCA）的原理、步骤、应用场景及实现方式，为读者提供了易懂的 PCA 入门指导。
